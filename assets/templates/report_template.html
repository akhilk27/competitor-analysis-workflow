Methodology

How this report was generated

Competitor Discovery: Google Search via Serper API using targeted site-specific queries for known {{ industry }} companies. Top {{ competitor_count }} unique results selected after deduplication and root-domain normalization.

Content Scraping: Homepage, pricing, about, and feature pages extracted via Firecrawl (clean markdown output; navigation and boilerplate filtered). Polite delays between requests; partial results used gracefully where pages were unavailable.

Analysis: All scraped content sent to Claude (Anthropic) in a single prompt to ensure consistent cross-competitor comparison — strengths, weaknesses, positioning, pricing, and market-level recommendations tailored specifically to {{ business_name }}.

Limitations: Based on publicly available website content as of {{ report_date }}. Pricing may not reflect enterprise tiers or negotiated rates. Comparison table dimensions are inferred from scraped content and may not capture all product capabilities.
